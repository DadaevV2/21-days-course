## 📖 О проекте

Это полноценное веб-приложение для личностного роста, построенное на принципах:

- **Научный подход** — каждая практика основана на исследованиях
- **Системность** — 21 день структурированной работы над собой
- **Глубокий анализ** — ИИ-анализ ваших отчётов
- **Персонализация** — выбор из 6 курсов

---

## ✨ Возможности

| Блок | Описание |
|------|----------|
| 🌅 **Утро** | Концепция + наука + практика + цитата |
| ☀️ **День** | Вопрос для рефлексии + микро-практика |
| 🌙 **Вечер** | Осознанный отчёт |
| 📊 **Аналитика** | 6 измерений вашего прогресса |
| 💡 **Рекомендации** | Персональные советы на основе анализа |
| 🎯 **6 курсов** | По 21 дню каждый |
